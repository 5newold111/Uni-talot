# シーズーうにの占いの館

深い夜のインクとアンティークな真鍮金に包まれた、本物のシーズー犬「**うに**」がいるタロット占いの館。
大アルカナ22枚で、あなたの今を静かに読み解きます。ブラウザだけで動作・インストール不要・完全オフライン。

> 「やっほー、来てくれて嬉しいワン。うにと一緒に視てくれる？……あ、いま さつまいも食べてたの。えへへ」― 占い師うに（女の子・食いしん坊・ちょっと甘えん坊）

## 遊び方

1. `index.html` をブラウザで開く（うにの館へ入る）
2. 占いたい **テーマ**（総合運・恋愛運・仕事運・金運）を選ぶ
3. **占い方（スプレッド）** を選ぶ
4. 願いを念じながら、裏向きの札を引く
5. 占い師うにが、引いた札を読み解いて鑑定してくれる

## デザイン

実在の占い・タロットサイトの手法を参考に、"AIっぽさ"を避けた editorial なビジュアルに設計しています（絵文字は不使用）。

- **Golden Thread Tarot / Labyrinthos** … 暗いマット地に金の細線（フィリグリー）と幾何学的な意匠
- **アストロロジー系タロットクロス** … 派手な金ではなくアンティークな真鍮金で上品に、星と月の縁取り
- **しいたけ占い（VOGUE）** … 明朝体の editorial な見出し、余白、テーマ別カラーコード
- **アール・ヌーヴォー（ミュシャ系）タロット** … アーチ型の装飾枠と飾り罫（フィリグリー）
- **ライダー＝ウェイト版** … ローマ数字のカード番号、ギルト縁、正逆位置の作法

素材（星空・アーチ枠・カード背面・テーマ図像・水晶・飾り罫）はすべて Canvas / SVG で生成し、うにの肖像はアーチ枠に納めています。

## 特徴

- 占い師「うに」の一貫した世界観 — 本物のうにの写真を肖像に、語りかけ・見立て・署名まで（女の子・さつまいも好き・ちょっと甘えん坊）
- 大アルカナ22枚を完全収録。各カードに「本質」＋テーマ別の正位置・逆位置の解釈
- 深い鑑定 — 各カードは「読み解き」＋「具体的アドバイス」の二段構成
- 引いたカードを一つの物語として関連づける総合鑑定文（流れ・全体トーン・締めの言葉）
- 3種類のスプレッド（ワンオラクル／スリーカード／状況スプレッド）
- シャッフルとランダムな正逆位置で毎回違う結果
- 水晶玉を覗き込む占い演出、星空アニメーション
- スマホ対応のレスポンシブ、`prefers-reduced-motion` 配慮

## ファイル構成

`index.html` の **1ファイルだけ** で完結（CSS・JavaScript・カードデータ・うにの写真をすべて内包）。
他のファイルは不要で、開くだけで動きます。

## 動作環境

モダンなブラウザ（Chrome / Firefox / Safari / Edge）。サーバー不要・他ファイル不要。
