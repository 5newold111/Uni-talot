# 🔮 シーズーうにの占いの館

紫のベルベットと金の星月に包まれた、子犬の占い師「**うに**」がいるタロット占いの館。
ブラウザだけで遊べます。インストール不要・完全オフライン動作。

> 「やっほー、来てくれて嬉しいワン！うにと一緒に視てくれる？……あ、いま さつまいも食べてたの。えへへ」― 占い師うに（女の子・食いしん坊・ちょっと甘えん坊）🐾🍠

## 遊び方

1. `index.html` をブラウザで開く（うにの館へ入る）
2. 占いたい **テーマ**（総合運・恋愛運・仕事運・金運）を選ぶ
3. **占い方（スプレッド）** を選ぶ
4. 心の中で願いを念じながら、裏向きのカードを引く
5. 占い師うにが、引いたカードを読み解いて鑑定してくれる

## 特徴

- 🐶 **占い師「うに」** の世界観 — 本物のうにの写真をマスコットに、語りかけ・見立て・肉球の署名まで一貫したブランディング（女の子・食いしん坊でさつまいも好き・ちょっと甘えん坊）
- 🟣 紫ベルベット × 金の星月をモチーフにした、館らしい配色と装飾（マスコット・ビネット・金の額縁）
- 🃏 **大アルカナ22枚** を完全収録。各カードに「本質」＋テーマ別の正位置・逆位置の解釈
- 🔮 **深い鑑定** — 各カードは「読み解き」＋「具体的アドバイス」の二段構成
- 🧵 引いたカードを **ひとつの物語として関連づける総合鑑定文**（流れ・全体トーン・締めの言葉）
- 🎯 テーマ別（総合・恋愛・仕事・金運）の解釈
- 🔀 3種類のスプレッド
  - **ワンオラクル** … 1枚で今のメッセージ
  - **スリーカード** … 過去・現在・未来
  - **状況スプレッド** … 現状・課題・アドバイス
- ✨ シャッフル＆ランダムな正逆位置で毎回違う結果
- 🌌 星空の演出とカードめくりアニメーション
- 📱 スマホ対応のレスポンシブデザイン

## ファイル構成

`index.html` の **1ファイルだけ** で完結しています（CSS・JavaScript・カードデータをすべて内包）。
他のファイルは不要なので、このファイルを保存してダブルクリックするだけで遊べます。

## 動作環境

モダンなブラウザ（Chrome / Firefox / Safari / Edge）。サーバー不要・他ファイル不要で、
`index.html` を開くだけで動きます。
