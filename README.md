# 🔮 ミスティック・タロット占い

ブラウザだけで遊べる、タロット占いゲームです。インストール不要・完全オフライン動作。

## 遊び方

1. `index.html` をブラウザで開く
2. 占いたい **テーマ**（総合運・恋愛運・仕事運・金運）を選ぶ
3. **占い方（スプレッド）** を選ぶ
4. 心の中で願いを念じながら、裏向きのカードを引く
5. 引いたカードの **正位置 / 逆位置** に応じた占い結果が表示される

## 特徴

- 🃏 **大アルカナ22枚** を完全収録。各カードに正位置・逆位置の意味
- 🎯 テーマ別（総合・恋愛・仕事・金運）の解釈
- 🔀 3種類のスプレッド
  - **ワンオラクル** … 1枚で今のメッセージ
  - **スリーカード** … 過去・現在・未来
  - **状況スプレッド** … 現状・課題・アドバイス
- ✨ シャッフル＆ランダムな正逆位置で毎回違う結果
- 🌌 星空の演出とカードめくりアニメーション
- 📱 スマホ対応のレスポンシブデザイン

## ファイル構成

`index.html` の **1ファイルだけ** で完結しています（CSS・JavaScript・カードデータをすべて内包）。
他のファイルは不要なので、このファイルを保存してダブルクリックするだけで遊べます。

## 動作環境

モダンなブラウザ（Chrome / Firefox / Safari / Edge）。サーバー不要・他ファイル不要で、
`index.html` を開くだけで動きます。

> タロットは未来を決めるものではなく、心を映す鏡です 🌙
