このページの使い方

Chrome 拡張機能をインストールしなくても、このページから URL を貼り付けるだけで 家具の 3D モデルを生成できます。

制約

  • サーバー側 HTML 解析の品質は拡張機能版より控えめ (og:image / og:title 中心)
  • JS で動的描画される SPA ページは情報が取れない
  • 寸法を自動抽出できない場合があるので、必要なら手動入力欄を使う
  • 3D 生成は MODEL_PROVIDER 設定に従う (デフォルト Tripo 有料 / 無料は Colab+TRELLIS)

API として使うとき

curl -X POST http://localhost:3000/api/process-url \
  -H "Content-Type: application/json" \
  -d '{"url":"https://www.nitori-net.jp/ec/product/12345/"}'
